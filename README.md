# 🏠 Boston House Price Prediction

Machine Learning project to predict median house prices in Boston using Linear Regression.

## 📊 Project Overview

This project demonstrates a complete end-to-end ML pipeline:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Outlier detection using IQR method
- Feature scaling with StandardScaler
- Linear Regression model training
- Comprehensive model evaluation

## 🛠️ Technologies Used

- Python 3.8+
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Jupyter Notebook

## 📁 Dataset

Boston Housing Dataset with 13 features:
- CRIM: Crime rate
- RM: Average number of rooms
- LSTAT: % lower status population
- MEDV: Median house value (target)
- And 9 more features
